# 头像设置

Valine 目前使用的是Gravatar 作为评论列表头像。

请自行登录或注册Gravatar，然后修改自己的头像。

评论的时候，留下在Gravatar注册时所使用的邮箱即可。

感谢gravatar.cat.net提供的镜像服务。

WARNING

如果你改了头像后发现没有更新，请不要慌张，因为gravatar.cat.net 有七天的缓存期，安静的等待吧~

目前有以下7种可选值:

参数值表现形式备注
空字符串''Gravatar官方图形
mm神秘人(一个灰白头像)
identicon抽象几何图形(根据邮箱或昵称生成)
monsterid小怪物
wavatar用不同面孔和背景组合生成的头像(根据邮箱或昵称生成)
retro八位像素复古头像(根据邮箱或昵称生成)
hide不显示头像
new Valine({
    ...
    avatar:'' // (''/mm/identicon/monsterid/wavatar/retro/hide)
});
